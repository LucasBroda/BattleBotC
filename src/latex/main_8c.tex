\doxysubsection{Detailed Description}
Programme principal pour la connexion et l\textquotesingle{}analyse radar dans le jeu Battle-\/C. 

Ce fichier contient la fonction principale qui établit une connexion avec le serveur, récupère les informations du monde et active un radar pour détecter les joueurs à proximité.
